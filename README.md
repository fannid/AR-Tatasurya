# AR-Tatasurya
### Project 3D AR Tatasurya

Aplikasi Media Pembelajaran sistem Tata Surya dengan Augmented Reality (Juni 2019)

Tools yang digunakan dalam mengembangkan Aplikasi :
1. Unity 2018.3.3f1
2. Vuforia
3. blender
4. Illustrator & CorelDraw
